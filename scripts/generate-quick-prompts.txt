# AI精灵图快速生成指南

## 推荐工具

### 1. DALL-E 3 (最简单)
- 网址: https://chat.openai.com/

### 2. Midjourney (高质量)
- 网址: https://www.midjourney.com/

### 3. Stable Diffusion (最佳控制)
- 网址: https://stablediffusionweb.com/

---

## 快速开始

### 步骤1: 复制粘贴提示词

选择你要的单位，复制对应的提示词

### 步骤2: 调整并生成

将提示词粘贴到AI工具中，调整细节，生成图像

### 步骤3: 保存到ai-sprites/目录

保存为PNG图片，命名:
- allied_soldier.png
- soviet_soldier.png
- allied_tank.png
- soviet_tank.png
等等...

### 步骤4: 运行后处理

```bash
node scripts/process-ai-sprites.cjs
```

## 提示词技巧

### 更好的像素化
- 添加 "sharp clean pixels, pixel-perfect"

### 更清晰的背景
- 添加 "on transparent background, alpha channel"

### 更好的方向
- 添加 "8 compass directions: north, northeast, east, southeast, south, southwest, west, northwest"

### 颜色约束
- 添加 "color palette limited"
