## 🎯 最佳方案：使用DALL-E生成单帧，再拼接

### 步骤1: 生成单个方向的精灵帧

使用以下提示词模板：

```
[具体单位描述], pixel art style, 64x64 pixels, top-down view from classic 1990s RTS game like Command & Conquer Red Alert 2, isometric perspective, single character/unit centered in frame, transparent background, sharp clean pixels, no anti-aliasing, limited color palette typical of 1990s games, high contrast
```

**关键要点**:
- `pixel art style` - 确保是像素风格
- `64x64 pixels` - 指定像素尺寸
- `transparent background` - 透明背景
- `top-down view` - 俯视角
- `Command & Conquer Red Alert 2 style` - 参考风格

## 🎬 生成动画帧的技巧

精灵图需要多个帧来显示动画效果。使用以下方法：

### 方法1: 分次生成，每帧调整

生成4次提示词，分别为：
```
# 帧0: 站立
[单位], pixel art, 64x64, standing pose, feet together

# 帧1: 左脚向前
[单位], pixel art, 64x64, walking animation frame 1, left foot forward, arms swinging

# 帧2: 跨步
[单位], pixel art, 64x64, walking animation frame 2, both feet apart, mid-stride

# 帧3: 右脚向前
[单位], pixel art, 64x64, walking animation frame 3, right foot forward, arms swinging opposite

# 帧4: 回到站立
[单位], pixel art, 64x64, walking animation frame 4, returning to standing position
```

### 方法2: 让AI生成精灵图序列

```
pixel art sprite sheet, US soldier walking animation, 4 frames in a row, 64x64 pixels each frame, top-down view, blue uniform, standing-walk1-walk2-walk3 sequence, transparent background, Red Alert 2 style, sharp pixels
```

## 📐 生成8个方向

对于每个单位，需要生成8个方向：
1. 北 (North) - 背对屏幕
2. 东北 (Northeast)
3. 东 (East) - 右侧
4. 东南 (Southeast)
5. 南 (South) - 面向屏幕
6. 西南 (Southwest)
7. 西 (West) - 左侧
8. 西北 (Northwest)

在提示词中添加：
```
facing [方向], [direction] orientation
```

## 💾 保存和使用

### 1. 保存文件名
```
单位精灵图保存为:
  public/assets/sprites/units/allied_soldier.png
  public/assets/sprites/units/soviet_soldier.png
  public/assets/sprites/units/allied_tank.png
  ...

建筑精灵图保存为:
  public/assets/sprites/buildings/allied_command.png
  public/assets/sprites/buildings/soviet_command.png
  ...
```

### 2. 精灵图布局要求

**单位精灵图** (256x512像素):
```
[帧0北][帧1北][帧2北][帧3北]
[帧0东北][帧1东北][帧2东北][帧3东北]
[帧0东][帧1东][帧2东][帧3东]
[帧0东南][帧1东南][帧2东南][帧3东南]
[帧0南][帧1南][帧2南][帧3南]
[帧0西南][帧1西南][帧2西南][帧3西南]
[帧0西][帧1西][帧2西][帧3西]
[帧0西北][帧1西北][帧2西北][帧3西北]
```

**建筑精灵图** (128x128像素):
```
单帧，直接使用
```

### 3. 处理AI生成的图片

如果AI生成的是单个图片：
1. 使用图片编辑工具（Photoshop, GIMP, Preview等）
2. 裁剪到正确尺寸
3. 排列成精灵图布局
4. 保存为PNG格式
